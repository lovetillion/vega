# vega
POC mainnet-js/libauth based adapter to trade Cauldron DEX Swaps

## Setup

* Make sure you have a Node.js and NPM environment setup
* Clone the repo
* ```cd vega```
* ```npm install```

## Instructions

* ```node vega.js [SEEDPHRASE]```

## Goal

Implement a trivial Swap transaction
